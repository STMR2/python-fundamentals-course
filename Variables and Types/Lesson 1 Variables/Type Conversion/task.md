# Data Types in Python

| Type | Description | Example |
|------|-------------|---------|
| `int` | Whole numbers | `42`, `-7` |
| `float` | Decimals | `3.14`, `-0.5` |
| `str` | Text | `"Hello"` |
| `bool` | True/False | `True`, `False` |
```python
print(type(42))       # <class 'int'>
print(type("Hello"))  # <class 'str'>
print(type(True))     # <class 'bool'>
```
