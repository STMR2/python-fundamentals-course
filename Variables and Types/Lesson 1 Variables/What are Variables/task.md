# What are Variables?

A **variable** is like a labeled box that stores data in your program.
In Python, you create a variable by assigning a value using the `=` operator.
```python
name = "Alice"
age = 25
height = 1.68
```

Python **automatically detects the type** — this is called **dynamic typing**.

### Variable Naming Rules
- Must start with a **letter** or **underscore** (`_`)
- Are **case-sensitive** (`age` ≠ `Age`)
- Can contain letters, numbers, and underscores
- Cannot be a reserved keyword (e.g., `for`, `if`, `class`)
